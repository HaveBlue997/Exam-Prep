# Quick Start Guide

## Starting the Server

1. Open Terminal
2. Run these commands:
   ```bash
   cd /Users/chadc/dev/GitRepo/Exam-Prep/Server
   npm start
   ```

3. You should see:
   ```
   Exam Prep Server Started
   Local: http://localhost:3000
   ```

## Access URLs

**From your Mac:**
- http://localhost:3000

**From your iPad (on same Wi-Fi):**
- http://192.168.7.238:3000
- *(Replace with your current IP if it changes)*

## How to Find Your IP Address

If your IP address changes, run:
```bash
ipconfig getifaddr en0
```

Or: System Settings > Network > Wi-Fi > Details

## Using the Quiz

1. Open the URL on your iPad
2. Click "Government"
3. Click "am gov quiz"
4. Fill out name and date
5. Answer questions
6. Click "Save Answers" - saves to server!
7. You can view saved answers from the Government class page

## Stopping the Server

Press `Ctrl+C` in the Terminal window.
